About Termux Fish

Termux Fish is a script to permanently install the Fish shell in Termux and enhance the appearance of your Termux homepage using Neofetch. With Fish shell, you get features like auto-completion, syntax highlighting, and many more. Here's how you can install it easily:

Installation Instructions

Run the following single command in your Termux terminal:

pkg update && pkg upgrade -y && pkg install git -y && git clone https://github.com/msn-05/termux-fish.git && cd termux-fish && chmod +x script.sh && ./script.sh

Once the script runs, exit Termux and reopen it to see the changes.

Steps to Install Manually

  1. Update Termux: pkg update && pkg upgrade -y
  2. Install Git: pkg install git -y
  3. Clone the repository: git clone https://github.com/msn-05/termux-fish.git
  4. Change to the cloned directory and give executable permissions: cd termux-fish && chmod +x script.sh
  5. Run the installation script: ./script.sh
  6. Exit Termux and enter again to see the changes.

If you want to revert back to your previous shell, run this command: chsh -s bash
